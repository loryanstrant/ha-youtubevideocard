# YouTube Video Card

Display YouTube videos and playlists directly in your Home Assistant dashboard.

## Features

- Play individual YouTube videos
- Play YouTube playlists
- Customizable video player controls
- Multiple aspect ratio options
- Autoplay, loop, mute options
- Start/end time controls

## Quick Start

Add manually to dashboard YAML:

```yaml
type: custom:youtube-video-card
video_id: dQw4w9WgXcQ
autoplay: false
controls: true
```
