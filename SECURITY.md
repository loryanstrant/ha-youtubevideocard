# Security Policy

## Supported Versions

We release patches for security vulnerabilities for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please send an email to the maintainer or create a private security advisory through GitHub.

**Please do not report security vulnerabilities through public GitHub issues.**

Include the following information in your report:
- Type of vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will respond to security issues within 48 hours and aim to release a fix within 7 days for critical vulnerabilities.

## Security Best Practices

When using this card:
1. Only use trusted video IDs and playlist IDs
2. Be cautious with autoplay features, especially with sound
3. Keep the card updated to the latest version
4. Review the card configuration for any suspicious content
5. If using an API key, ensure it's properly restricted

## Third-Party Dependencies

This card uses the YouTube IFrame API, which is hosted by Google. By using this card, you are subject to:
- [YouTube Terms of Service](https://www.youtube.com/t/terms)
- [Google Privacy Policy](https://policies.google.com/privacy)

## Disclosure Policy

- Security vulnerabilities will be disclosed after a fix is available
- Credit will be given to reporters who identify vulnerabilities
- We follow responsible disclosure practices
